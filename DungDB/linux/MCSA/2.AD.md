# Note lại thực hành triển khai Active Directory Domain Service trên Windows server 2019

## Chuẩn bị

Cài đặt máy ảo Windows server 2019 standard GUI

Cấu hình IP tĩnh

Update server 

Cấu hình:

- 8GB RAM

- 50GB disk

- 4 core

- IP: 172.16.70.223
